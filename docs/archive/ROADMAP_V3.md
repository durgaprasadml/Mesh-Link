# Roadmap: Version 3.0 (V3)

With Mesh Link V2.0 achieving GA, architectural planning for V3 commences immediately.

## Strategic Pillars for V3
1. **Mesh-over-Internet Bridges:** Allowing isolated localized meshes to bridge across the globe if at least one node in each mesh has an active cellular/Starlink connection.
2. **Federated AI:** Distributing Lightweight LLM inference tasks across idle nodes in the mesh to provide disaster-response intelligence without internet connectivity.
3. **Desktop Companion Apps:** Expanding the mesh topology to include macOS, Windows, and Linux nodes via standard BLE/Wi-Fi adapters.
4. **Offline Topographic GIS:** Integrating MBTiles to render offline maps with live tracking of mesh nodes for search-and-rescue (SAR) operations.
5. **Satellite / LoRa Integration:** Hardware hooks for USB-C connected LoRa radios to exponentially increase node hop distance.
