# Changelog

All notable changes to this project will be documented in this file.
The format is based on Keep a Changelog.

## [1.0.0] - 2026-07
### Added
- Enterprise-grade BLE Mesh routing and relay architecture.
- Wi-Fi Direct hybrid transport for media and large payloads.
- SQLCipher encrypted Room database.
- Elliptic Curve Diffie-Hellman (ECDH) key exchange.
- Jetpack Compose UI with Material 3 dynamic theming.
- Full Android 13-17 compatibility including Foreground Service policies.
- Extensive test coverage (120+ Unit/Integration tests).

### Security
- R8/ProGuard obfuscation enabled for release builds.
- Strict Linting enabled for release gates.
