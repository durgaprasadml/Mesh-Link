# General Availability (GA) Declaration

Mesh Link Version 2.0 is officially designated as **General Availability (GA)**.

## Certification Milestones Achieved
- **RC1:** Memory leaks, SQLite locking, and Coroutine concurrency stabilized.
- **RC2:** Battery consumption minimized; BLE and Wi-Fi Direct MTU/buffer throughput maximized.
- **RC3:** Multi-OEM (Xiaomi, Samsung, Pixel) edge cases mitigated. Android 14+ Foreground Service policies cemented.
- **RC4:** Chaos-tested against 100-node network floods, massive broadcast storms, and sudden hardware failure.
- **RC5:** Enterprise MDM, Google Play, and GDPR privacy certifications passed.

The `main` branch is now locked for V2.0.0. All future development shifts to the V3 roadmap and `develop` branch.
