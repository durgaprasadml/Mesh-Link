# Release Notes: Version 2.0.0 (GA)

## Mesh Link V2.0.0 is Live!
This marks the official production release of the fault-tolerant, decentralized, offline communication platform.

## Features at a Glance
- **Decentralized Messaging:** End-to-end encrypted messaging via Bluetooth Low Energy (BLE) and Wi-Fi Direct. Zero internet required.
- **Media Transfer:** Send images, voice notes, and videos across the mesh.
- **Multi-Hop Routing:** Messages dynamically bounce across devices up to 15 hops away to reach their destination.
- **Enterprise Security:** AES-256-GCM encryption with ECDH session negotiation.

## Stability Guarantee
This version is certified to run on Android 13 through Android 16, surviving aggressive battery optimizers and massive network floods without draining your battery.
