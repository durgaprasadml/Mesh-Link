# Mesh Link V2.0 Release Notes

**Welcome to Mesh Link V2.0!**

This major release solidifies Mesh Link as a production-grade, fault-tolerant communication platform designed for extreme disaster and enterprise environments.

## What's New & Hardened
- **Zero-Cloud Resiliency:** Certified offline operation. The app does not ping external servers, ensuring functionality in total internet blackouts.
- **Enterprise MDM Ready:** Fully compatible with Android Work Profiles, Samsung Knox, and Microsoft Intune for managed deployments.
- **Battery Optimization:** Relentlessly tuned background service Wakelocks ensure the mesh stays alive without draining your battery in the field.
- **Android 14/15 Support:** Fully modernized for the latest edge-to-edge UI and strict foreground service policies.

*This version contains no API breaking changes and maintains 100% backward compatibility with V1.X nodes.*
